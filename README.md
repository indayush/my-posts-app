## run the development server:

```bash
npm run dev
# or
yarn dev
```

## Build Next App

First, run the development server:

```bash
npm run build
# or
yarn build
```

Build Output in .next folder

## Build Next App as Static Webpage App

First, run the development server:

```bash
npm run export
# or
yarn export
```

Build Output in out folder
